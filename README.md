# Hamza's Portfolio - Orderflow Landing Page

A simple, responsive landing page for showcasing trading and orderflow analysis expertise.

## Features

- Responsive design that works on desktop and mobile
- Smooth scrolling navigation
- Hero section with call-to-action
- About, Projects, and Contact sections
- Basic contact form with JavaScript validation

## Usage

1. Open `index.html` in your web browser to view the landing page.
2. Customize the content in `index.html` to match your information.
3. Modify styles in `style.css` for branding and design changes.
4. Add more interactivity in `script.js` if needed.

## Files

- `index.html`: Main HTML structure
- `style.css`: CSS styles for layout and design
- `script.js`: JavaScript for interactivity

## Customization

- Replace the placeholder background image in the hero section with your own.
- Update the project cards with real projects.
- Add your contact information and social links.

## Troubleshooting

- If images don't load, ensure paths are correct.
- For deployment, upload all files to a web server.
- Test on multiple devices for responsiveness.
